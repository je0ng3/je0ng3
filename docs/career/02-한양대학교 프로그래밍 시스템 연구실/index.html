Summary

연구실 인턴 | LLM을 사용한 코드 자동 생성 프로그램 구현을 통한 모델 성능 검사 (2024.03.02 - 2024.08.30 ) | Python, Hugging Face, PyTorch, TensorFlow

  • 자연어 처리 기술을 활용한 코드 자동 생성 프로그램 개발
  • Earley 파서를 직접 구현하여 생성 코드의 문법적 타당성 보장
  • Top-k 토큰 중 최적 후보를 선택해 생성 정확도 향상
  • 구조적 분석 및 검증 자동화 경험과 모델에 따른 출력 결과 비교를 통한 LLM 성능 평가 경험

Tasks

  • 자연어 명령어를 입력받아 코드로 자동 변환하는 시스템 개발
  • Hugging Face 기반 LLM을 활용한 코드 스니펫 생성 기능 구현
  • Earley 파서 구현을 통한 실시간 문법 오류 검출 시스템 개발. 코드 생성 시 토큰 단위로 문법 검사 수행하여 오류 사전 방지
  • 생성된 코드의 컴파일 성공 여부를 통한 자동화된 정확도 평가 로직 구현

Reflection

LLM이 코드를 생성하는 과정을 토큰 단위로 살펴보며, 코드가 제대로 생성되지 않거나 컴파일 에러가 발생하는 경우의 원인을 분석하는 과정이 좋았습니다. 코드를 단계별로 나눠서 확인하고, 렉싱이나 파싱 오류를 하나씩 잡아가며 구현하는 과정이 마치 디버깅하듯 느껴져서 흥미로웠습니다.
이번 프로젝트에서는 성능 평가를 중심으로 구현했지만, 다음에는 코드 생성의 정확도를 높이기 위한, 문제를 해결하고 생산성을 높기기 위한 일까지 해보고 싶습니다.
